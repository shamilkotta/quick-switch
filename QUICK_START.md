# Quick Start - Development

## Setup

1. Go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" → Select `quick-switch` folder

## Test

- Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
- Or click the extension icon in toolbar

## Development

**After making changes:**
- Reload extension: `chrome://extensions/` → Click refresh icon on Quick Switch
- Test: Press shortcut again

**Debug:**
- Right-click extension icon → "Inspect popup" (opens DevTools console)

**Files:**
- `popup.html` / `popup.css` / `popup.js` - Main popup UI
- `options.html` / `options.js` - Settings page
- `manifest.json` - Extension config
